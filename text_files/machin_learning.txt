Machin learning basics

Instead of writing detailed rules for a computer to follow, in machine learning you provide data and let the system find patterns on its own. For example, if you want a computer to recognize spam emails, you don’t write all possible spam rules—instead, you train a model using many examples of spam and non-spam emails. The system then learns how to classify new emails.

There are three main types of machine learning:

1. Supervised Learning
In this type, the model is trained using labeled data. This means each example comes with a correct answer. For instance, predicting house prices based on features like size, location, and number of rooms.

2. Unsupervised Learning
Here, the model works with unlabeled data and tries to find hidden patterns or groupings. An example is customer segmentation, where users are grouped based on behavior without predefined categories.

3. Reinforcement Learning
In this approach, an agent learns by interacting with an environment and receiving rewards or penalties. It improves its performance over time by maximizing rewards. This is commonly used in robotics and game-playing AI.
